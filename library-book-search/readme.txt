This plugin will list library book search.

== Description ==

Library Book Search can manage based on library book search which will be based on book name, author, publisher, price, book rating.

== Installation ==

1. Upload the entire `library-book-search` folder to the `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' menu in WordPress.

You will find 'Book' menu in your WordPress admin panel.

2. Book > Shortcode Reference page copy the shortcode and paste page/post.
